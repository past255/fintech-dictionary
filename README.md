# Fintech & Product Metrics Library

Справочник ключевых показателей финтеха и продукт-менеджмента в одном HTML-файле. Без сборки, установки и серверов — достаточно открыть `index.html` в браузере.

## Что это

81 показатель, разложенный по 8 тематическим категориям. Для каждого — краткое описание, формула или логика расчёта и «смысл и ценность»: ради чего этот показатель вообще считают.

Библиотека задумана как быстрая шпаргалка: когда нужно освежить память по конкретной метрике или быстро сверить определение в рабочем обсуждении.

## Категории

| Категория | О чём |
|---|---|
| **Финансы** | Банковская и корпоративная рентабельность, процентная маржа, регуляторные риск-метрики и показатели P&L |
| **Юнит-экономика** | Здоровье бизнес-модели на уровне клиента и подписки, стоимость и окупаемость привлечения |
| **Риски** | Кредитный риск: вероятность дефолта и ожидаемые потери |
| **Клиентский опыт** | Лояльность, удовлетворённость, юзабилити и качество работы поддержки |
| **Надёжность** | Стабильность, скорость и качество технической части продукта |
| **Вовлечённость** | Как пользователь живёт в продукте и возвращается ли |
| **Воронка** | Конверсии на каждом этапе пользовательского пути |
| **Дискавери** | Фреймворки продуктового ресёрча и приоритизации гипотез |

## Возможности

- Поиск по аббревиатуре, названию или описанию
- Фильтрация по категории в один клик
- Раскрывающиеся карточки с формулой и пояснением
- Адаптивная вёрстка под десктоп и мобильные

## Как запустить

Откройте `index.html` в любом современном браузере. Tailwind, Alpine.js и шрифт Inter подгружаются с CDN — локальная установка не нужна.

## Стек

- **Tailwind CSS** — стили через CDN
- **Alpine.js** — интерактивность и состояние
- **Inter** — шрифт

Один HTML-файл — всё внутри, без `node_modules` и сборки.
